# YDFDA.org Color Scheme

## Brand Colors

### Primary Teal/Green
| Name | Hex | RGB | Usage |
|------|-----|-----|-------|
| Primary Teal | `#4e7f76` | rgb(78, 127, 118) | Main brand color, buttons, section backgrounds |
| Dark Teal | `#336666` | rgb(51, 102, 102) | Headings, secondary sections, footer buttons |
| Medium Teal | `#476d67` | rgb(71, 109, 103) | Navigation links |

### Accent Orange
| Name | Hex | RGB | Usage |
|------|-----|-----|-------|
| Primary Orange | `#d16103` | rgb(209, 97, 3) | CTA buttons, links, border accents, site slogan |
| Light Orange | `#d48355` | rgb(212, 131, 85) | Hover states, secondary accents, menu borders |
| Burnt Orange | `#C97E51` | rgb(201, 126, 81) | Homepage feature box background |

## CSS Custom Color Variables (Suggested)

```css
:root {
  /* Brand Colors */
  --brand-teal: #4e7f76;
  --brand-teal-dark: #336666;
  --brand-orange: #d16103;
  --brand-orange-light: #d48355;
  
  /* Backgrounds */
  --bg-white: #ffffff;
  --bg-light: #f5f5f5;
  
  /* Text */
  --text-primary: #000000;
  --text-secondary: #333333;
  --text-muted: #999999;
  
  /* Borders */
  --border-color: #dddddd;
  
  /* UI States */
  --link-color: #428bca;
  --success: #5cb85c;
  --warning: #f0ad4e;
  --danger: #d9534f;
  --info: #5bc0de;
}
```
